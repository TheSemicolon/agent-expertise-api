────────────────────────────────────────────────────────────────────────────────
2. Required Secrets
────────────────────────────────────────────────────────────────────────────────

Two Secrets must exist in the {{ .Release.Namespace | quote }} namespace BEFORE
the pods become Ready. The chart does NOT create these — the operator manages
them out-of-band (via SealedSecrets, External Secrets Operator, sops, etc.).

  • {{ .Values.auth.secretName | quote }}
        Mounted on the API container via envFrom. Must contain at minimum:
          AUTH__APIKEY                       (LocalDev/ApiKey/Hybrid only)
          Auth__Oidc__Issuers__N__GroupToTenantMapping__<group-uuid>
                                             (per OIDC issuer if not in values.yaml)

  • {{ .Values.postgres.secretName | quote }}
        Used by the postgres StatefulSet and the API connection string.
        Must contain:
          POSTGRES_PASSWORD
          POSTGRES_USER       (optional — defaults to "expertise")

────────────────────────────────────────────────────────────────────────────────
3. Production checklist
────────────────────────────────────────────────────────────────────────────────
{{- if eq .Values.auth.mode "Oidc" }}
{{- if not .Values.auth.oidc.issuers }}

  WARNING: auth.mode=Oidc but auth.oidc.issuers is empty.
           The API will fail startup with a clear error message.
           Populate at least one issuer (Entra, Authentik, etc.) in values.yaml
           and `helm upgrade` to recover.
{{- else }}

  ✓ Auth mode: Oidc with {{ len .Values.auth.oidc.issuers }} issuer(s) configured.
{{- end }}
{{- else }}

  WARNING: auth.mode={{ .Values.auth.mode | quote }}.
           Modes other than "Oidc" are permitted only when
           ASPNETCORE_ENVIRONMENT=Development. The API's startup guard will
           hard-fail in any other environment.
{{- end }}
{{- if not .Values.forwardedHeaders.trustedCidr }}

  WARNING: forwardedHeaders.trustedCidr is empty.
           Audit logs will record the ingress pod IP rather than the real
           client IP. Set this to your cluster pod CIDR (and any other proxy
           networks) to fix attribution. Typical value:
             forwardedHeaders:
               trustedCidr:
                 - "{{ .Values.postgres.podCidr }}"   # cluster pod CIDR
{{- else }}

  ✓ ForwardedHeaders trusts {{ len .Values.forwardedHeaders.trustedCidr }} CIDR(s).
{{- end }}
